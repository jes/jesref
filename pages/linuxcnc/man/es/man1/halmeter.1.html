NOMBRE

halmeter - observa los pines, señales y parametros de HAL

SINOPSIS

halmeter [-s] [pin|sig|param name] [-g X-positon Y-position [Width] ]

DESCRIPCION

halmeter se utiliza para observar pines, señales, o parametros HAL (Capa de abstraccion de hardware). Sirve para el mismo proposito que un multimetro cuando se trabaja en sistemas fisicos.

OPCIONES

pin name

mostrar el pin HAL name.

sig name

mostrar la señal HAL name.

param name

mostrar el parametro HAL name. Si no se especifica pin, sig o param , la ventana comienza en blanco y el usuario debe seleccionar un elemento para observar.

-s

ventana pequeña, no interactiva, que debe utilizarse con pin, sig, o param para seleccionar el elemento a mostrar. Se muestra el nombre del elemento en la barra de titulo en lugar de la ventana, y no hay botones "Seleccionar" o "Salir". Practico cuando se requieren muchas medidas en un espacio reducido.

-g

Posicion de la geometria. Permite especificar la posicion inicial y, opcionalmente, el ancho del multimetro. Referenciado desde la parte superior izquierda de la pantalla en unidades de pixel. Util cuando se requieren muchos multimetros en un script, sin solapar uno encima del otro.

USO

A menos que se especifique -s, hay dos botones, "Seleccionar" y "Salir". "Seleccionar" abre un cuadro de dialogo para seleccionar el elemento (pin, señal o parametro) a ser observado. "Exit" termina el multimetro.

El dialogo de seleccion tiene botones "Aceptar", "Aplicar" y "Cancelar". "Aceptar" muestra el elemento seleccionado y cierra el dialogo. "Aplicar" muestra el elemento seleccionado pero mantiene abierto el dialogo de seleccion. "Cancelar" cierra el dialogo sin cambiar el elemento mostrado.

EJEMPLOS

halmeter

Abre una ventana de medidor, sin mostrar nada inicialmente. Use el boton "Seleccionar" para elegir un elemento a observar. No termina hasta que se cierre la ventana.

halmeter &

Abre una ventana de medidor, sin mostrar nada inicialmente. Use el boton "Seleccionar" boton para elegir un elemento. Se ejecuta en segundo plano dejando el shell libre para otros comandos.

halmeter pin parport.0.pin-03-out &

Abre una ventana de multimetro, mostrando inicialmente el pin HAL parport.0.pin-03-out. El boton "Seleccionar" se puede utilizar para mostrar otros elementos. Se ejecuta en segundo plano.

halmeter -s pin parport.0.pin-03-out &

Abre una pequeña ventana de multimetro, que muestra el pin HAL parport.0.pin-03-out. El elemento mostrado no se puede cambiar. Se ejecuta en segundo plano.

halmeter -s pin parport.0.pin-03-out -g 100 500 &

Abre una pequeña ventana que muestra el pin HAL parport.0.pin-03-out y la coloca 100 pixeles a la izquierda y 500 pixeles hacia abajo desde la parte superior de la pantalla. El elemento mostrado no se puede cambiar. Se ejecuta en segundo plano.

halmeter -s pin parport.0.pin-03-out -g 100 500 400 &

Abre una pequeña ventana que muestra el pin HAL parport.0.pin-03-out y la coloca 100 pixeles a la izquierda y 500 pixeles hacia abajo desde la parte superior de la pantalla. El ancho sera de 400 pixeles (270 es el valor predeterminado). El elemento mostrado no se puede cambiar. Se ejecuta en segundo plano.
