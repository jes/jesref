NOMBRE

axis-remote - Interface remota AXIS

SINOPSIS

axis-remote OPCIONES|NOMBREFICHERO

DESCRIPCION

axis-remote es un pequeño script que activa comandos en una GUI AXIS en ejecucion.
Use axis-remote --help para mas informacion.

OPCIONES

--ping, -p

Chequea si se esta ejecutando AXIS.

--reload, -r

Hace que AXIS recargue el archivo actual.

--clear, -c

Hace que AXIS limpie el area grafica.

--quit, -q

Salida de AXIS.

--help, -h, -?

Muestra la lista de parametros validos para axis-remote.

--mdi COMANDO, -m COMANDO

Ejecuta el COMANDO en MDI.

NOMBREFICHERO

Carga el fichero de codigo G NOMBREFICHERO.

VEA TAMBIEN

axis(1)

Los manuales de LinuxCNC y HAL tienen mucha mas informacion disponible.
Puede encontrarlos en /usr/share/doc/linuxcnc/.
