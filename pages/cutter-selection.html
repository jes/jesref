Best all-rounder

For use in both material removal and profiling, use the 8 mm 4-flute cutter.

Using the 8 mm cutter for material removal is 92% as fast as using the 6 mm cutter.

Using the 6 mm cutter for profiling is only 80% as fast as using the 8 mm cutter.

Faster spindle?
